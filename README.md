# YummyBox

Plateforme web Django pour partager et découvrir des recettes gourmandes, avec système de catégories, avis utilisateurs, et gestion complète des recettes.

## Fonctionnalités

- Liste et détails des recettes
- Ajout, modification, suppression (avec permissions)
- Système d’évaluation et commentaires
- Gestion des catégories

## Installation

1. Clone le projet  
`git clone https://github.com/TON_USERNAME/NOM_DU_REPO.git`

2. Installe les dépendances  
`pip install -r requirements.txt`

3. Lance le serveur  
`python manage.py runserver`
